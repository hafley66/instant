## Solution/Approach
- Cmd-Tab (lib.rs): set_switcher_visible() keyed off the window FOCUS event (focus→Regular/skip_taskbar(false); blur→Accessory/skip_taskbar(true)); demote-on-blur lands cleanly (already resigned active). JS blur-hide timer 500→120ms (main.ts).
- Safe-boot (state.ts SAFE_BOOT): skips reading localStorage so corrupt dockJSON can't jam. Triggers: Cmd+Shift+R (main.ts keymap), tray "Safe Reopen" (lib.rs eval sessionStorage+reload), ?safe URL. Tray "Reset All State…" hard-wipes with confirm.
- Logging: Rust log_append/log_path/log_reveal → app_data_dir/instant.log (com.instant.summon). Frontend logLine(); showError + console.error mirrored to it.
- AI master switch: state.ts aiEnabled (default true, persisted). Off → both launch pickers drop agent entries (main.ts ~1419/1743), openWorktreeDefault opens shell. Toggle: agents-editor checkbox + tray "Toggle AI Integrations" (emits toggle-ai → store).
- rg highlight: markClickMatches/markTokenInNode wrap query occurrences in <mark class=rg-mark> + <span class=rg-idx> index. CRITICAL FIX: dropped shiki syntax-coloring on rg HIT ROWS (highlightClickHits removed) — shiki split rows into per-token spans so a punctuated/multi-token query like "markTokenInNode(root" spanned multiple text nodes and never matched. Plain text = one node per row → match always found. markClickMatches runs synchronously now (no shiki dependency), idempotent (unwraps prior marks).
- Badge final CSS (.rg-idx): plain inline <span>, NO display, NO position/transform/flex — just padding:2px 4px; border-radius:999px; vertical-align:super; black bg/white text. (Every flex/inline-flex/position/sup attempt visually split the number from the circle in WebKit; padding+vertical-align on a bare inline span was the fix.)
- Back-nav: previewOrigin map + reactdock activatePreviewPanel(key); preview meta shows "← back" to the originating rg panel (delegated handler).
- DOM cmd+click (main.ts wireDomCmdClick): capture-phase doc handler runs dispatchClick on free text in .fs-preview/.rg-panel (terminals self-handle), word via caretRangeFromPoint.
- Quoted-string smart selection: quotedSpanAt(text,col) returns unquoted contents of '…'/"…"/`…`; wired into terminal link provider (whole quoted span = one link, inner \S+ fragments skipped), wordAt, domWordAt. Handles paths with spaces.
- Preview line view (renderPathInto): now shiki per-line highlighted; active line translucent yellow + left accent (was navy/white that killed colors); meta filename bold+escaped; theme-flip re-renders line previews too.
- Dockview double-blue: scoped Luna-blue active-tab to .dv-active-group (xp light+dark); inactive-group selected tab muted; base theme --dv-inactivegroup-visiblepanel-tab-* → head-bg.
