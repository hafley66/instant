## Solution/Approach
- Titlebar drag + double-click maximize: own it in JS (`mousedown` on `.title-bar`, `e.detail===2` => toggleMaximize else startDragging), since `data-tauri-drag-region` only matches the exact target.
- Window edge resize: tao's macOS `drag_resize_window` returns NotSupported for ALL dirs, so `startResizeDragging` is a no-op on mac. Replaced with manual pointer-capture resize (setSize/setPosition, screenX/Y are logical px).
- Auto-resume harness sessions: Rust `harness::harness_session(tool,cwd)` — claude=newest `~/.claude/projects/<cwd:/→->/*.jsonl` stem; opencode=`SELECT id FROM session WHERE directory=? AND time_archived IS NULL ORDER BY time_updated DESC` (rusqlite). `WtAgent.resume` flag + `autoResume` config (default true). Resolve by id (not bare --continue: claude errors w/o session; opencode --continue is global).
- Pin tabs (⌘⇧P, 📌 title via reactdock setTermTitle, float left via moveTermPanel+reflowPinnedTabs), reopen closed tab (⌘⇧T, closedTabs stack; close_pty only detaches so reattach works).
- react-table: `treetable.tsx` headless <TreeTable> (emits .dtable markup), `tablepanels.tsx` TmuxPanelV2/WorktreesPanelV2 + bridge pattern (main.ts owns derivation/handlers), registered as panels tmux2/worktrees2.
- XP polish: explicit color/bg on .panel-scroll/.dtable (dockview host doesn't inherit --panel-fg), .s-proc teal in dark xp, skin-scoped listview gradient headers+separators+blue selection; XP rail icons from @react95/icons (only its png/ is usable; copied 7 to public/icons/, PanelDef.iconUrl => <img class=ai-img>).
- Resizable rail: drag divider `#actbar-resize` on rail right edge (big mode), persists sidebarWidth, applyRailWidth().
- Bug fixes: tab nav (⌘1-9/next/prev) walks active group's full visual panel list (groupPanelIds/activePanelId/focusPanelById) so it reaches v2 tool panels; ⌘W closes dockview active panel (closeActivePanel) not stale store.active; moveTermPanel clamps index (out-of-range threw gridview "invalid location", broke new-tab); xterm ⌘C copies selection (paste left to xterm native — handling it double-pasted).

## Key Insights
- Tauri mac is half-finished for borderless windows: startResizeDragging no-op (tao NotSupported), data-tauri-drag-region only matches exact target. Reimplement chrome in JS.
- close_pty only drops the in-proc pty handle; tmux session survives => reopen reattaches.
- claude/opencode session stores: claude flat jsonl per cwd-encoded dir; opencode SQLite `session.directory`.
- @react95/icons published form only ships png/ (css/components broken, exports blocks deep png imports) => copy needed pngs to public/.
- dockview: api.activeGroup/activePanel exist; group.panels is visual tab order; moveTo index must be < group size or it throws "invalid location".
- PanelDef.component (React) is used directly by dockComponents; component panels skip PanelDef.onShow (pool-adoption only) so v2 panels do mount useEffect -> bridge.onShow.
