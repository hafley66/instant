## Solution/Approach

ALL code agents in worktrees (user directive, in memory) with FIRST ACTION `git merge main --no-edit` (worktrees fork from origin/main which lags local main -- caused stale-base conflicts early). Fable merges, runs gates, prunes worktree+branch. Kill agents past ~200k tokens and relaunch remainder as fresh single-package agents (salvage completed commits).
